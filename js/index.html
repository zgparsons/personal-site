Javascript

Javascript is a high-level language which is supported by all modern web browsers. Along with HTML and CSS it is one of the core technologies of the web.

Javascript can be written right into an HTML document and will run automatically as the page is loaded. It can do this because browsers have a built in Javascript Engine. Originally these were only created for web browsers but now javascript can be run outside the browser by using a runtime environment like Node.js.
