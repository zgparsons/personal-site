Knowledge Module One

Content Management Systems

What is a Content Management System?

A content management system is a software application that we use to publish digital content. Content can be added, updated and deleted via the CMS (and typically an easy to use User Interface, or UI) rather than via files. Website content will typically include text primarily, and then also embedded graphics, images & photos, videos, audio files, maps, and other applications (which use different types of program code), which will all be displayed to the user. A successful CMS must therefore be able to store and deliver all of these file types.

According to user statistics (and Wikipedia) Wordpress is the most popularly used Content Management System. It is used by over 28% of all websites, and 59% of sites known to be using a CMS. Joomla and Drupal are two other market leaders.

Common Features of CMS

  • Design is seperate from content
  • SEO-Friendly URL structuring
  • Shared resources
  • Regular updates - features & security
  • Mobile ready
  • Archive capabilities
  • Database driven
  • Remote access
  • Easy to learn/use - many tutorials & walkthroughts available online
  • Cost-effective (Wordpress is open source)
  • Easy to link to CRM (or ecommerce, or other) systems

CMS Advantages over Static Websites

  • No need to learn HTML, CSS, Javascript etc.
  • Usually there is version control built into CMS
  • CMS allows for different levels of editing permissions
  • Can be easier to create a unified look & feel, via using themes & plugins
  • Added functionality is usually a lot easier to integrate
  • CMS will have a WYSIWYG (What You See is What You Get) editor
